## **내 풀이**  
--- 

```python
def solution(array, commands):
    answer = []	

    for i in commands:
        c1  = i[0]
        c2 = i[1]
        c3 = i[2]
        
        sliced = array[c1-1:c2]
        sliced.sort()
        ans = sliced[c3-1]
        answer.append(ans)
        
    return answer
```

#### **코드 해설**  
1. commands 안의 리스트의 첫번째~3번째 원소를 각 변수에 저장한다. 
2. array에서 c1-1 (인덱스는 0부터) 부터 c2까지 슬라이싱해 sliced 변수에 저장한다.
3. sliced를 오름차순으로 정렬한 뒤 c3-1번째 숫자를 answer에 추가해 반환한다.
