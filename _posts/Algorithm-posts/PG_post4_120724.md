## **내 풀이**  
---  

```python
str1, str2 = input().strip().split(' ')
print(str1,str2,sep="")
```


#### **코드 해설**  
print의 sep 파라미터를 사용해 값을 구분할 때 넣을 문자를 지정할 수 있다.  

```python
sep="" # 값을 붙여서 출력
sep='\n' # 값들을 줄바꿈해서 출력 
sep='\t' # 값 사이를 tab으로 구분
```
