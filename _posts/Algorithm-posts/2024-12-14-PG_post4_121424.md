## **내 풀이**  
--- 

```python
def solution(n):
    a = ''
    for i in range(n):
        if i % 2 == 0:
            a += '수'
        else:
            a += '박'
    return a
```

#### **코드 해설**  
1. 0~n까지 순회하며 짝수일때 '수'를, 홀수일때 '박'을 변수에 추가한다.
