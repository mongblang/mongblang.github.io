## **내 풀이**  
--- 

```python
def solution(a, b):
    answer = 0
    for i in range(len(a)):
        answer+=a[i]*b[i] 
    return answer
```

#### **코드 해설**  
1. a의 길이만큼 순회하며 i번째 a와 i번째 b의 곱을 answer에 더한 후 반환한다. 

&nbsp;  


## **다른 풀이**
---

```python  
def solution(a, b):
    return sum([x*y for x, y in zip(a,b)])
```

#### **코드 해설**  
1. zip 함수를 사용해 a와 b를 합친 후 순회하며 곱한 값의 합을 더하여 반환한다.
