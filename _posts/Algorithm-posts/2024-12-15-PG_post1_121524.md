## **내 풀이**  
--- 

```python
def solution(n):
    answer = [i for i in range(1,n+1) if n%i == 1]
    return min(answer)
```

#### **코드 해설**  
1. n을 i로 나눈 나머지가 1인 경우 i를 answer에 추가한다.
2. answer 리스트의 최소값을 반환한다. 

&nbsp;  


## **다른 풀이**
---

```python  
def solution(n):
    return [x for x in range(1,n+1) if n%x==1][0]
```

#### **코드 해설**  
1. 풀이는 위와 동일하지만 min 대신 리스트의 첫번째 값 (가장 작은 값)을 바로 반환한다.
