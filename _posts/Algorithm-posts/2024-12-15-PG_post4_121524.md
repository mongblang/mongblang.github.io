## **내 풀이**  
--- 

```python
def solution(s):
    cnt_p = 0
    cnt_y = 0
    
    for i in s.lower():
        if i == 'p':
            cnt_p += 1
        elif i == 'y':
            cnt_y += 1
    if cnt_p == cnt_y:
        return True
    else:
        return False
```

#### **코드 해설**  
1. p와 y의 개수를 셀 변수를 정의한다.
2. s를 모두 소문자로 변경한 후 순회하며 각 문자가 p거나 y면 만들어둔 변수의 개수를 1씩 증가시킨다.
3. p의 개수와 y의 개수가 같으면 True를, 다르면 False를 반환한다.

def numPY(s):
