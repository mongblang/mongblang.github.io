## **내 풀이**  
--- 

```python
def solution(numbers):
    answer = 0
    for i in range(10):
        if i not in numbers:
            answer+=i     
    return answer
```

#### **코드 해설**  
1. 0~9까지 순회하며 i가 numbers에 없으면 i를 더한 값을 반환한다.

## **다른 풀이**

def solution(numbers):

1. number의 합을 0~9까지의 합인 45에서 빼서 존재하지 않은 수의 합을 바로 반환한다.
